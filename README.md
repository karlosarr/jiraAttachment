# jiraAttachment

Jira Attachment file with nodejs.
This project sending create file to Attachment file in issue.

## Installation
Need [Jira](https://www.atlassian.com/) 

Use [NodeJS](https://nodejs.org/es/) to install jiraAttachment.

```bash
npm i
```

## Usage

```bash
npm start
```
